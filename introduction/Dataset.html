Dataset

The Datasets used for training our models will be provided by the Open Graph Benchmark (OGB). For each Dataset, the following files will be provided:

  1. Graph Lables.

  2. Number of Nodes per Graph.

  3. Number of Edges per Graph.

  4. Edges between Nodes.

  5. Node Features

  6. Edge Features

Node Features

The Node Features will be displayed through a 9-dimensional Index List, which corresponds to:

  1. Atom Number (1 to 119) (H is excluded, so C is 5 not 6)

  2. Chirality (Unspecified, Tetrahedral Clockwise(CW), Tetrahedral Counter Clockwise(CCW), Other and Misc)

  3. Degree (0 to 10 and misc)

  4. Formal Charge (-5 to 5 and misc) (so an index of 5 would be a charge of 0)

  5. Number of H Atoms (0 to 8 and misc)

  6. Number of radicals (0 to 4 and misc)

  7. Hybridization (SP, SP2, SP3, SP3D, SP3D2, misc)

  8. Is Aromatic (False, True)

  9. Is in Ring (False, True)

Edge Features

The Edge Features will be displayed through a 3-dimensional Index List, which corresponds to:

  1. Bond Type (Single, Double, Triple, Aromatic, Misc)

  2. Bond Stereo (None, Z, E, Cis, Trans, Any)

  3. Is Conjugated (False, True)
